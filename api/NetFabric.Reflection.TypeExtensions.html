ImplementsInterface(Type, Type, out Type[]?)

Determines whether the specified Type implements a specified interface and, if so, provides information about generic type arguments.

public static bool ImplementsInterface(this Type type, Type interfaceType, out Type[]? genericArguments)

Parameters

type Type

The Type to check for interface implementation.

interfaceType Type

The Type of the interface to check for implementation.

genericArguments Type[]

When the method returns true, this parameter contains an array of Type objects representing the generic type arguments that make the implementation of the specified interface. If the type does not implement the interface or if the interface is non-generic, this parameter is set to null.

Returns

bool

true if the specified Type implements the specified interface; otherwise, false.
