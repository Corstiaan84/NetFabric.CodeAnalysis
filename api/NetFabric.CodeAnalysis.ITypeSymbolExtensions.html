IsAsyncEnumerable(ITypeSymbol, Compilation)

Checks if the provided typeSymbol can be used as a source in an await foreach statement, indicating whether it is an async enumerable.

public static bool IsAsyncEnumerable(this ITypeSymbol typeSymbol, Compilation compilation)

Parameters

typeSymbol ITypeSymbol

The ITypeSymbol to be checked for async enumerability.

compilation Compilation

The Compilation representing the current compilation context.

Returns

bool

true if the typeSymbol is an async enumerable; otherwise, false.

Remarks

This method examines the provided typeSymbol to determine if it can be used as a source in an await foreach statement, indicating whether it is an async enumerable. To be considered an async enumerable, the type should support asynchronous iteration, typically by implementing the IAsyncEnumerable<T> interface or providing a suitable asynchronous GetAsyncEnumerator method.

If the type is an async enumerable, it is considered an async source for an await foreach statement.
