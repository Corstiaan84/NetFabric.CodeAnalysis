GetExtensionMethodsWithName(Compilation, ITypeSymbol, string)

Retrieves a collection of extension methods within the specified compilation that are applicable to the given typeSymbol and have the specified name.

public static IEnumerable<IMethodSymbol> GetExtensionMethodsWithName(this Compilation compilation, ITypeSymbol typeSymbol, string name)

Parameters

compilation Compilation

The Compilation containing the types and methods to search in.

typeSymbol ITypeSymbol

The ITypeSymbol representing the type to which extension methods should be applicable.

name string

The name of the extension method to search for.

Returns

IEnumerable<IMethodSymbol>

A collection of IMethodSymbol representing extension methods that match the criteria. An empty collection is returned if no matching extension methods are found.

Remarks

This method searches for extension methods based on their name and applicability to the specified typeSymbol. Extension methods are methods that are marked as static and defined in non-nested, non-generic, static classes.
